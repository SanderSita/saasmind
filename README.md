# 🧠 Saasmind

[![Live Demo](https://img.shields.io/badge/Live%20Demo-saasminder.com-success)](https://saasminder.com)

**Saasmind** is an AI-powered chat assistant that remembers your project's context, so you don't have to repeat yourself.

Instead of starting every conversation from scratch, Saasmind builds an understanding of your project, documentation, decisions, and workflows. Ask questions, brainstorm ideas, generate content, or get coding assistance, all with responses tailored to your project.

### Features

- Persistent project memory
- AI chat with full project context
- Understands your documentation and notes
- More relevant AI responses
